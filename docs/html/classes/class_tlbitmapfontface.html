TLBitmapFontFace¶

Inherits: TLFontFace

Category: Core

Brief Description¶

An AngelCode Bitmap Font Generator bitmap font for drawing text.

Properties¶

int texture_flags O: 2048

Description¶

TLBitmapFontFace have limited shaping support.

TLBitmapFontFace doesn’t support OpenType features.
